# 1.2 Go支持的平台与架构
## Go 支持平台：
- Linux
- Windows
- Mac OS X（也称为 Darwin）

## Go 支持架构：
- Linux 2.6+：amd64、386 和 arm 架构
- Mac OS X (Snow Leopard + Lion) ：amd64 和 386 架构
- Windows 2000+：amd64 和 386 架构
